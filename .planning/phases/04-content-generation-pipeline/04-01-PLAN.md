       - Constructor takes `api_key: str`, creates `openai.AsyncOpenAI(api_key=api_key)`
       - `complete()`: calls `client.chat.completions.create()`. Maps response to CompletionResponse. Catches `openai.RateLimitError` -> `RateLimitError`. Catches status 500/503 -> `ServiceUnavailableError`.
       - `complete_structured()`: uses `response_format={"type": "json_schema", "json_schema": {"name": "response", "schema": schema}}`. Parses JSON from response content, validates with response_model.
       - `stream()`: calls with `stream=True`, iterates async chunks, yields StreamChunk.
       - Default model: `"gpt-4o"`

    6. Create `src/deckforge/llm/adapters/gemini.py` - GeminiAdapter(LLMAdapter):

       - `complete()`: calls `model.generate_content_async()`. Maps to CompletionResponse. Handles google.api_core.exceptions for rate limit (429) and server errors (500/503).
       - `complete_structured()`: sets `generation_config={"response_mime_type": "application/json", "response_schema": schema}`. Parses JSON, validates.
       - `stream()`: calls with `stream=True`, yields chunks.
       - Default model: `"gemini-2.0-flash"`

       - `stream()`: POST with `stream: true`, reads NDJSON lines, yields StreamChunk.

    1. Create `src/deckforge/llm/router.py`:
       - `LLMRouter` class:
         - Constructor takes `adapters: dict[str, LLMAdapter]`, `fallback_chain: list[str]`
         - `_resolve_provider(model: str | None) -> str`: maps model name prefix to provider ("claude" -> "claude", "gpt" -> "openai", "gemini" -> "gemini", default -> first in fallback_chain)
         - `async def complete(messages, model=None, **kwargs) -> CompletionResponse`: resolves provider, tries it, then falls back through chain on RateLimitError or ServiceUnavailableError. Raises AllProvidersFailedError if all fail.
         - `async def complete_structured(messages, model=None, response_model=None, **kwargs) -> BaseModel`: same routing + fallback logic
         - `async def stream(messages, model=None, **kwargs) -> AsyncIterator[StreamChunk]`: same routing (no fallback mid-stream)
       - `create_router(llm_config: LLMConfig | None = None, settings: Settings | None = None) -> LLMRouter`:
         - If llm_config has api_key, create single adapter for that provider with user's key
         - If llm_config has base_url, pass to adapter constructor (Ollama case)
         - If no llm_config, create adapters for all providers with non-None keys from settings
         - Build fallback_chain from settings.LLM_FALLBACK_CHAIN
         - Return configured LLMRouter
