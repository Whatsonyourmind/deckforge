Build the 4-stage content generation pipeline (intent parser, outliner, slide writer, cross-slide refiner) and wire it into the POST /v1/generate endpoint with SSE progress streaming.

Purpose: This is the core intelligence of DeckForge -- transforming a natural language prompt into a complete, coherent presentation IR. The pipeline replaces the stub generate_content worker task with real LLM orchestration.

Output: `src/deckforge/content/` package with pipeline stages + prompt templates, `/v1/generate` endpoint, SSE streaming endpoint, and updated worker task.
