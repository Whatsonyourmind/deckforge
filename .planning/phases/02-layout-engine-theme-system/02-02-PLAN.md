<objective>
Build the complete theme system: 15 curated YAML themes with 3-tier design tokens, variable resolution with cycle detection, brand kit overlay with protected keys, WCAG AA contrast validation, and per-slide-type slide masters.

Purpose: Themes control every visual property of a rendered slide. The layout engine (Plan 03) and PPTX renderer (Phase 3) both consume ResolvedTheme to determine colors, fonts, sizes, and spacing. Without themes, no slide can be styled.

Output: Six production modules in `src/deckforge/themes/`, 15 YAML theme files, and comprehensive unit tests.
</objective>
