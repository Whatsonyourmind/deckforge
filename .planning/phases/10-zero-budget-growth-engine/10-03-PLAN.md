<!-- DeckForge API endpoints that integrations call -->

POST /v1/render
- Body: Presentation IR JSON (schema_version, metadata, theme, slides)
- Query: output_format=pptx|gslides
- Response: { download_url, slide_count, quality_score, file_size_bytes }

POST /v1/generate
- Body: { prompt: string, slide_count: int, theme: string }
- Response: SSE stream -> final { download_url, slide_count, quality_score }

GET /v1/themes
- Response: [{ name, description, version }]

GET /v1/slide-types
- Query: category=universal|finance
- Response: [{ name, category, description, element_types }]

GET /v1/capabilities
- Response: { slide_types, themes, chart_types, output_formats }

GET /v1/pricing
- Response: { tiers: [...], x402: { render, generate } }

POST /v1/estimate
- Body: Presentation IR JSON
- Response: { credits, usd_equivalent }

Headers: Authorization: Bearer dk_live_xxx
Base URL: https://api.deckforge.io (or DECKFORGE_API_URL env var)
