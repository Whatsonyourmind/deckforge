       - Build a list of dicts with Task, Start, Finish from `chart_data.tasks`.

       - If mid_values provided, add marker diamonds at midpoints.
       - Use `go.Bar(orientation='h')` with base parameter for range positioning.

       **sensitivity.py** - `SensitivityChartRenderer(StaticChartRenderer)`:
       - Annotated heatmap: `go.Heatmap(z=data, x=col_values, y=row_values)` with `texttemplate` showing cell values.
       - X-axis title from col_header, Y-axis title from row_header.
       - Colorscale from theme positive/negative colors.

       **funnel.py** - `FunnelChartRenderer(StaticChartRenderer)`:
       - `go.Funnel(y=chart_data.stages, x=chart_data.values, marker=dict(color=colors))`.

       - base_value defaults to 0 if not provided. Offsets calculated as value - base_value.
       - `fig.update_layout(barmode='overlay')`.

       **sunburst.py** - `SunburstChartRenderer(StaticChartRenderer)`:
       - `go.Sunburst(labels=chart_data.labels, parents=chart_data.parents, values=chart_data.values, marker=dict(colors=colors))`.
