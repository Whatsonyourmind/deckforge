    Hand-write core IR types matching the Python Pydantic models (these will be replaced by auto-generated types when OpenAPI generation runs, but provide immediate type safety):
    - PresentationIR: schema_version, metadata, brand_kit, theme, slides, generation_options
    - PresentationMetadata: title, purpose, audience, confidentiality, language
    - SlideInput: discriminated union by slide_type covering all 32 types
    - ElementInput: discriminated union by type (heading, subheading, body_text, bullet_list, chart, table, image, etc.)
    - ChartInput: discriminated union by chart_type
    - BrandKit: colors, fonts, logo
    - GenerationOptions: target_slide_count, density, chart_style, quality_target

    - InsufficientCreditsError: not enough credits

    sdk/src/client.ts -- DeckForge class:
    ```typescript
    export class DeckForge {
      private baseUrl: string;
      private apiKey: string;

      constructor(opts: { apiKey: string; baseUrl?: string }) { ... }

      // Direct API methods (SDK-01)
      async render(ir: PresentationIR): Promise<RenderResult> { ... }
      async renderToBuffer(ir: PresentationIR): Promise<ArrayBuffer> { ... }
      async estimate(ir: PresentationIR): Promise<CostEstimate> { ... }
      async getJob(jobId: string): Promise<JobStatus> { ... }

      // Deck operations
      decks = {
        list: (opts?) => Promise<DeckListResponse>,
        get: (id) => Promise<DeckDetail>,
        getIR: (id) => Promise<PresentationIR>,
        delete: (id) => Promise<void>,
        appendSlides: (id, slides) => Promise<DeckDetail>,
        replaceSlide: (id, index, slide) => Promise<DeckDetail>,
        reorderSlides: (id, order) => Promise<DeckDetail>,
        retheme: (id, theme) => Promise<DeckDetail>,
      };

      // Builder entry point
      get presentations() { return { create: PresentationBuilder.create } }

      // Generation with streaming
      generate = {
        start: (prompt, opts?) => Promise<JobStatus>,
        stream: (prompt, opts?) => AsyncGenerator<ProgressEvent>,
      };
    }
    ```

    Internal _fetch method handles:
    - Authorization header with API key
    - Content-Type: application/json
    - Error response parsing into typed errors
    - Retry on 429 with Retry-After header

    Create sdk/src/builder/presentation.ts -- PresentationBuilder:
    - Static create(title: string): returns new PresentationBuilder
    - .theme(themeId: string): returns new builder with theme set
    - .metadata(meta: Partial<PresentationMetadata>): returns new builder
    - .brandKit(kit: BrandKit): returns new builder
    - .addSlide(slide: SlideInput): returns new builder with slide appended
    - .addSlides(slides: SlideInput[]): returns new builder with all slides appended
    - .generationOptions(opts: Partial<GenerationOptions>): returns new builder
    - .build(): validates and returns PresentationIR (throws if invalid -- e.g. no slides)
    - .render(client: DeckForge): calls client.render(this.build())
    - .estimate(client: DeckForge): calls client.estimate(this.build())

    All methods return new builder instances (immutable pattern).
