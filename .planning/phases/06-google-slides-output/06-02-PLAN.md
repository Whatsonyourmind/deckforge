    **charts.py:** Create `SheetsChartBuilder` class:

    ```python
    class SheetsChartBuilder:
        """Creates Google Sheets charts for embedding in Slides."""

        def __init__(self, credentials, presentation_id: str):
            self.sheets_service = build("sheets", "v4", credentials=credentials)
            self.drive_service = build("drive", "v3", credentials=credentials)
            self.presentation_id = presentation_id
            self.spreadsheet_id: str | None = None
            self._sheet_index = 0  # Track worksheet index for each chart
    ```

    Methods:
    - `create_spreadsheet(self) -> str` -- Creates temp spreadsheet named "DeckForge Charts - {presentation_id}". Stores and returns spreadsheet_id.
    - `add_chart(self, chart_data, chart_type: str, title: str) -> int | None`:
      1. Determine if chart_type is Sheets-compatible. Map DeckForge ChartType to Sheets chart type:
         - bar/stacked_bar/grouped_bar/horizontal_bar -> "BAR" (with stackedType for stacked, orientation for horizontal)
         - line/multi_line -> "LINE"
         - area/stacked_area -> "AREA" (with stackedType for stacked)
         - pie/donut -> "PIE" (donut is pie with a pieHole)
         - scatter/bubble -> "SCATTER"
         - combo -> "COMBO"
      2. For unsupported types, return None (caller falls back to static image)
      3. Add a new worksheet to the spreadsheet for this chart: `addSheet` request via sheets batchUpdate
      4. Write chart data to the new worksheet: categories in row 1, series data in subsequent rows. Use `spreadsheets.values().update()`
      5. Create EmbeddedChart via `addChart` request in sheets batchUpdate. Set chart spec with basicChart type, domains (row 1), series (rows 2+), legend, axis titles.
      6. Parse response to get chartId: `response["replies"][0]["addChart"]["chart"]["chartId"]`
      7. Increment `_sheet_index`
      8. Return chartId

    - `get_slides_chart_request(self, chart_id: int, page_id: str, x: float, y: float, w: float, h: float, linked: bool = True) -> dict`:
      Returns a CreateSheetsChartRequest dict:
      ```python
      {
          "createSheetsChart": {
              "objectId": generate_object_id(),
              "spreadsheetId": self.spreadsheet_id,
              "chartId": chart_id,
              "linkingMode": "LINKED" if linked else "NOT_LINKED_IMAGE",
              "elementProperties": make_element_properties(page_id, x, y, w, h)
          }
      }
      ```

    - `get_static_image_request(self, image_url: str, page_id: str, x: float, y: float, w: float, h: float) -> dict`:
      For unsupported chart types, creates a CreateImageRequest with a static PNG URL.

    **Handling static chart fallback:**
    When `add_chart()` returns None (unsupported type):
    1. Use the existing `StaticChartRenderer` from `deckforge.rendering.chart_renderers` to render chart data to PNG bytes
    2. Upload PNG to S3 via `deckforge.workers.storage.upload_file()` to get a publicly accessible URL
    3. Use `get_static_image_request()` with that URL

    The GoogleSlidesRenderer orchestrator handles this flow: it checks if SheetsChartBuilder.add_chart() returns a chartId or None, and branches accordingly.

    ```python
    def cleanup_temp_spreadsheets(drive_service, spreadsheet_ids: list[str]) -> list[str]:
        """Delete temporary spreadsheets. Returns list of IDs that failed to delete."""
        failed = []
        for sid in spreadsheet_ids:
            try:
                drive_service.files().delete(fileId=sid).execute()
            except Exception as e:
                logger.warning("Failed to delete temp spreadsheet %s: %s", sid, e)
                failed.append(sid)
        return failed
    ```

    Endpoints:
    - `GET /v1/auth/google/authorize` -- Returns JSON with `authorization_url`. Accepts optional `state` query param. Generates the Google OAuth consent URL via GoogleOAuthHandler.get_authorization_url(). Requires valid API key authentication (same CurrentApiKey dependency as render).

    - `GET /v1/auth/google/callback` -- OAuth callback. Receives `code` and `state` query params from Google redirect. Exchanges code for tokens via GoogleOAuthHandler.exchange_code(). Stores encrypted refresh_token associated with the API key in the database (add a `google_refresh_token` column to the `api_keys` table, or create a new `google_oauth_tokens` table with api_key_id FK). Returns JSON with `{"status": "connected", "scopes": [...]}`.

    - `DELETE /v1/auth/google` -- Revokes Google access. Deletes stored refresh token for the API key. Requires CurrentApiKey.

    Note on token storage: Use a simple new column `google_refresh_token: str | None` on the existing ApiKey SQLAlchemy model. Encrypt at rest using Fernet with a key derived from DECKFORGE_SECRET_KEY (add SECRET_KEY to Settings if not present). For MVP, storing as plain text is acceptable with a TODO for encryption.

    ```python
    def render_pipeline(
        presentation: Presentation,
        output_format: str = "pptx",
        credentials=None,
    ) -> bytes | GoogleSlidesResult:
        theme_registry = ThemeRegistry()
        text_measurer = TextMeasurer()
        layout_engine = LayoutEngine(text_measurer, theme_registry)

        layout_results = layout_engine.layout_presentation(presentation)
        theme = theme_registry.get_theme(presentation.theme, presentation.brand_kit)

        if output_format == "gslides":
            from deckforge.rendering.gslides import GoogleSlidesRenderer
            renderer = GoogleSlidesRenderer()
            return renderer.render(presentation, layout_results, theme, credentials)
        else:
            renderer = PptxRenderer()
            return renderer.render(presentation, layout_results, theme)
    ```
