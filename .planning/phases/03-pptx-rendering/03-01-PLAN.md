<objective>
Build the core PPTX rendering engine that converts IR slides with layout positions into a valid PowerPoint file. This plan creates the PptxRenderer orchestrator and all element renderers (text, table, image, shape, data visualization), plus transition and speaker notes support.

Purpose: This is the foundational rendering layer that all PPTX output depends on. Without it, the API returns stub JSON instead of actual presentations.
Output: `src/deckforge/rendering/` package with PptxRenderer and element renderers, producing valid .pptx bytes from any IR + LayoutResult input.
</objective>

       - Footer row (if TableContent.footer_row): bold, top-border emphasis, surface background
       - Highlight rows (if TableContent.highlight_rows): accent background fill
       - Column widths: distribute proportionally based on header text lengths
       - Cell text alignment: numbers right-aligned, text left-aligned (detect by attempting float parse)
       - Font: body_family, caption scale size

         - "contain": calculate aspect-ratio-preserving dimensions, center within position bounds

       - Add via `shapes.add_picture(BytesIO(image_bytes), left, top, width, height)`

    Create `src/deckforge/rendering/pptx_renderer.py` with `PptxRenderer` class:

    ```python
    class PptxRenderer:
        def render(
            self,
            presentation: Presentation,
            layout_results: list[LayoutResult],
            theme: ResolvedTheme,
        ) -> bytes:
            """Render a complete PPTX from IR + layout results + theme."""
    ```

    Implementation:
    1. Create `pptx.Presentation()`, set `slide_width=Inches(13.333)`, `slide_height=Inches(7.5)` (PPTX-08)
    2. Find blank layout via `get_blank_layout(prs)`
    3. For each LayoutResult in layout_results:
       a. `slide = prs.slides.add_slide(blank_layout)`
       b. Set background: look up `theme.slide_masters.get(slide_type_str)` for master background color. If not found, use `theme.colors.background`. Call `set_slide_background(slide, bg_color)`.
       c. Iterate `layout_result.slide.elements`. For each element:
          - Get position from `element.position` (set by LayoutEngine). If None, skip.
          - Look up renderer from `ELEMENT_RENDERERS[element.type]` (or element.type.value if enum)
          - Call `renderer.render(slide, element, element.position, theme)`
       d. Apply transition: if `layout_result.slide.transition` is not None and not Transition.NONE, call `set_transition(slide, transition)` (PPTX-06)
       e. Apply speaker notes: if `layout_result.slide.speaker_notes` is not None, access `slide.notes_slide.notes_text_frame`, clear it, set text (PPTX-07)
    4. Save to BytesIO, return bytes
