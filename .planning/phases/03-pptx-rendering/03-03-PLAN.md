       ```python
           upload_file(s3_client, bucket, file_key, pptx_bytes, "application/vnd.openxmlformats-officedocument.presentationml.presentation")
       ```

       ```python
       def pptx_to_thumbnails(

           max_slides: int = 5,
           dpi: int = 150,
       ) -> list[bytes]:
           """Convert PPTX to PNG thumbnails via LibreOffice headless.

           Falls back to a simple placeholder if LibreOffice is not available.
           """
       ```

       Implementation:
       - Write pptx_bytes to a temp file
       - Run `libreoffice --headless --convert-to pdf --outdir {tmpdir} {pptx_path}` with timeout=30s
       - If libreoffice is not available (FileNotFoundError), fall back to a Pillow-generated placeholder PNG with text "Preview unavailable — LibreOffice not installed"
       - Convert PDF to PNG using pdf2image: `convert_from_path(pdf_path, first_page=1, last_page=max_slides, dpi=dpi)`
       - If pdf2image is not available, fall back to placeholder
       - Return list of PNG bytes (one per slide, up to max_slides)
       - Clean up temp files in finally block

       Also add a sync helper `pptx_to_thumbnails_fallback` that creates a simple Pillow image with slide number text — this is the fallback when LibreOffice is not installed (development environments, CI without LibreOffice).
