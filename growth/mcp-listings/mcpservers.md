## Full Description (for submission form)

DeckForge is an MCP server that enables AI agents and developers to generate executive-ready presentations via API. It supports 32 slide types across universal and finance categories, 15 curated themes, 24 chart types (including native editable and static Plotly charts), and a specialized finance vertical for institutional-grade output.

**Key Features:**
- 32 slide types (23 universal + 9 finance-specific)
- 15 curated themes with brand kit overlay support
- 24 chart types including waterfall, sankey, gantt, heatmap
- Finance vertical: comp tables, DCF summaries, deal overviews
- TypeScript SDK (@deckforge/sdk) with fluent builder API
- x402 machine payments for per-call USDC billing
- Dual transport: stdio (local) and streamable-http (production)

**MCP Tools:**
1. `render` -- Render Presentation IR into PowerPoint
2. `generate` -- Generate presentation from natural language
3. `themes` -- List available themes
4. `slide_types` -- List available slide types
5. `cost_estimate` -- Estimate rendering cost
6. `pricing` -- Get pricing tiers and rates
